System LED lamp control interface, driven by contactless chip.


Parameters
[in]lednum- The LED name
[in]mode- High 4 represents the control mode, such as open, off, sparking;The lower 28 bits represent the interval in flashing mode, in the unit of 1ms
Returns
0 -—success
-1 -—failed, You can obtain the error code according to errno.
