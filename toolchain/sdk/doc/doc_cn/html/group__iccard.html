◆ iccard_exchange_apdu()

int iccard_exchange_apdu ( int  fd,
uint32_t  sendlen,
const void *  senddata,
uint32_t *  recvlen,
void *  recvdata 
)

在读卡器和iccard之间交换apdu包（读卡器和IC卡进行交互信息）

参数
[in]fd
  • 打开读卡器的句柄, 函数 iccard_open_cardslot 的返回值
[in]sendlen
  • APDU发送数据的长度
[in]senddata
  • APDU发送数据的值
[out]recvlen
  • APDU接收数据的长度
[out]recvdata
  • APDU接收数据的值
返回

  • 0 成功
  • -1 失败，请检查“errno”的实际错误码
  • ENXIO 卡未在卡槽
  • EINVAL 无效的参数
  • ETIME 超时
  • ECOMM 奇偶校验错误或edc错误
  • EPROTO 无效信息
  • EACCES 卡片需要先重置

◆ iccard_power_up()

int iccard_power_up ( int  fd,
int  vcc,
int  mode,
int *  len,
void *  atr 
)

IC卡上电

参数
[in]fd- 打开读卡器的句柄, 函数 iccard_open_cardslot 的返回值
[in]vcc- VCC卡的值:
  • CVCC_1_8_VOLT
  • CVCC_3_VOLT
  • CVCC_5_VOLT
[in]mode- 操作卡类型:
  • ISO_MODE --— iso7816 类型
  • EMV_MODE --— EMV specification 类型
  • ISO_19200 --— 快速卡 iso7816 类型, 19200bps
  • ISO_38400 --— 快速卡 iso7816 类型, 38400bps
[out]len- atr信息长度
[out]atr- atr信息
返回

0 -成功
-1 -失败，请检查“errno”的实际错误码
ENXIO -卡未在卡槽
EINVAL -无效的参数
ETIME -读卡超时
ECOMM -奇偶校验错误或edc错误
EPROTO -来自iccard的无效信息
