uint8_t  keytype,
const void *  key,
const void *  uid 
)

读取mifare ultralight卡的某页内容，一次读取4个页内容

参数
[in]fdmif_open函数返回所打开设备的文件号
[in]blockno要认证的数据块号，对于S50卡，取值范围为0~63；对于S70卡，取值范围为0~255
[in]keytype要认证的密钥类型，取值如下：
'A'：认证 A 密钥
'B'：认证 B 密钥
[in]key6字节长度的密钥信息
[in]uid4字节的UID信息
返回值
0密码验证操作成功
-1操作失败，请通过errno获取具体的错误码：
EACCES ：感应区内不是mifare one卡
EPERM ：验证失败
EPROTO ：卡片返回数据不符合mifare one协议
EINVAL ：参数错误
ECOMM ：卡片返回的数据奇偶错或者CRC校验错
EIO ：读卡芯片错误
ETIME ：超时返回
其他值 ：其它错误码

◆ mifare_one_poll()

int mifare_one_poll ( int  fd,
uint8_t  mode,
void *  uid,
uint8_t *  cardtype 
)

检测感应区内是否存在mifare one卡，若有则激活

参数
[in]fdmif_open函数返回所打开设备的文件号
[in]mode轮询模式，取值如下： 0：仅轮询所有处于IDLE状态的卡片 1：轮询所有处于IDLE或者HALT状态的卡片
[out]uid卡片返回的5字节UID信息，4字节UID+BCC
[out]cardtype返回的mifare one卡片类型，可能取值如下：
MIF_M1_S50 ：mifare one S50 卡
MIF_M1_S70 ：mifare one S70 卡
MIF_PRO_S50 ：兼容T=CL协议和S50协议的TypeA卡
MIF_PRO_S70 ：兼容T=CL协议和S70协议的TypeA卡
返回值
0寻卡操作成功
-1操作失败，请通过errno获取具体的错误码：
ENODATA ：感应区内无 mifare one 卡
EMULTIHOP ：感应区内有多卡
EPROTO ：卡片返回数据不符合 ISO14443 协议
EBADE ：卡片返回的数据长度不正确
EINVAL ：参数错误
ECOMM ：卡片返回的数据奇偶错或者 CRC 校验错
EIO ：读卡芯片错误
ETIME ：超时返回
其他值 ：其它错误码
