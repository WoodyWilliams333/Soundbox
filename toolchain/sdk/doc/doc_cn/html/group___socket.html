◆ net_select()

int net_select ( int  nfds,
void *  readfds,
void *  writefds,
void *  exceptfds,
struct timeval *  timeout 
)

用来统一监视多个文件描述符

参数
[in]nfds- 集合中所有文件描述符的范围，即所有文件描述符的最大值加1
[in]readfds- 用来检查可读性的一组文件描述符集合
[in]writefds- 用来检查可写性的一组文件描述符集合
[in]exceptfds- 用来检查是否有异常条件出现的文件描述符集合
[in]timeout- 超时时间
返回值
>0- 被监视的文件描述符有变化
-1- 发生错误
=0- 超时
注解
输入参数readfds、writefds、exceptfds数据结构如下:
typedef struct sci_fd_set {

} sci_fd_set;

◆ net_socket()

int net_socket ( int  domain,
int  type,
int  protocol 
)

创建一个套接字，同时指定协议和类型

参数
[in]domain- 协议族
[in]type- 套接字的类型
[in]protocol- 指定传输协议
返回值
>=0- 成功，返回非负套接字描述符
-1- 失败，请通过errno获取具体的错误码
注解
[1] 协议族有AF_INET、AF_INET6、AF_LOCAL、AF_ROUTE等等
[2] 套接字的类型有SOCK_STREAM、SOCK_DGRAM、SOCK_SEQPACKET、SOCK_RAW等等
[3] 传输协议常用的有IPPROTO_TCP、IPPTOTO_UDP，通常设置为0使用默认的值
目前套接字支持wifi和无线网络，使用前可以先切换网络
