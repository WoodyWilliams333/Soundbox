long os_msg_queue_receive ( QueueHandle_t  xQueue,

)

消息队列接收

参数
[in]xQueue- 队列handle
[in]ms- (单位ms)读取时发生阻塞最大超时时间 0：不阻塞 0xFFFFFFFF：永久阻塞
[out]pvBuffer- 获取到的消息队列项
返回
  • -1 调用失败
  • 0 接收失败
  • 1 接收成功
