◆ kb_get_key_info()

int kb_get_key_info ( uint32_t  timeout_ms,
key_info_t *  info 
)

获取键值

参数
[in]timeout_ms- 等待按键超时时间,单位:ms; 传入-1表示阻塞式等待
[out]info- 按键的详细信息，包含按键状态和按键上报时的tick时钟，参见 key_info_t
返回
0 - 表示超时,没有获取到按键信息
>0 - 表示按键值
注解
如果没有按键,该函数会等待超时后再返回; 未获取键值时,该函数返回0; 如果不关心key_info，可传入NULL

◆ kb_select_key_func()

int kb_select_key_func ( int  kv,
int  fidx 
)

选择按键功能

注解
某些按键支持多个功能，通过此函数切换
参数
[in]kv- 需要切换的按键的键值，第一或第二功能键值均可
[in]fidx- 0:第一功能 1:第二功能
返回
0 - 成功
-1 - 不支持多功能
