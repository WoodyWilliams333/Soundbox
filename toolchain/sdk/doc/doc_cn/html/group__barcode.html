◆ BarcodeMode

enum BarcodeMode

该枚举列出了扫码接口支持的模式.

枚举值
BAR_MODE_CONTINUE 

支持一维码、二维码、连续扫描码

BAR_MODE_ONCE 

支持一维码、二维码，只扫描一次后打开

BAR_MODE_MOTIONDETECT 

不受支持的。支持一维码和二维码。当检测到图像亮度变化时发出扫描码，否则与BAR_MODE_CONTINUE相同

BAR_MODE_ONED_ONLY 

不支持BAR_MODE_CONTINUE。只扫描一个维度代码，否则与BAR_MODE_CONTINUE相同

BAR_MODE_MOTIONDETECT_MINI 

不支持此参数。与BAR_MODE_MOTIONDETECT相同，不推荐使用

int bar_ioctl ( int  fd,

)

设置条码参数

参数
[in]fd- 描述符
[in]cmd- BAR_IOCTL_SET_SOUND 设置扫码声音 arg:1 有声 arg:0 无声
BAR_IOCTL_SET_READ_CALLBACK 设置扫码成功后的回调函数，形式为 bar_callback_t
[in]arg- 与 cmd 对应

-1 - 失败
int fd;
int res;
fd = bar_open("/dev/barcode", BAR_MODE_ONCE); //open后只扫描一次
res = bar_ioctl(fd, BAR_IOCTL_SET_PREVIEW, TRUE); //TRUE:打开预览 FALSE:关闭预览
if (!res) {
//---- display code ----
lcd_flip();
}
bar_close(fd);
