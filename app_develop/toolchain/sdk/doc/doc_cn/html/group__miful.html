◆ mifare_ultralight_poll()

int mifare_ultralight_poll ( int  fd,
uint8_t  mode,
void *  uid 
)

检测感应区内是否存在mifare ultralight卡，若有则激活

参数
[in]fdmif_open函数返回所打开设备的文件号
[in]mode轮询模式，取值如下： 0：仅轮询所有处于IDLE状态的卡片 1：轮询所有处于IDLE或者HALT状态的卡片
[out]uid卡片返回的7字节UID信息
返回值
0寻卡操作成功
-1操作失败，请通过errno获取具体的错误码：
ENODATA ：感应区内无mifare ultralight卡
EMULTIHOP ：感应区内有多卡
EPROTO ：卡片返回数据不符合ISO14443协议
EBADE ：卡片返回的数据长度不正确
EINVAL ：参数错误
ECOMM ：卡片返回的数据奇偶错或者CRC校验错
EIO ：读卡芯片错误
ETIME ：超时返回
其它值 ：其它错误码
