函数说明

◆ nfc_led_control()

int nfc_led_control ( int  lednum,
int  mode 
)

系统LED灯控制接口，由非接芯片驱动.

参数
[in]lednum- LED名称号
[in]mode- 高4位代表控制模式，如打开，关闭，闪灯；低28位代表闪灯模式下的间隔时间，单位为1ms
返回
0 -—成功
-1 -—失败, 可根据errno获取错误码.
