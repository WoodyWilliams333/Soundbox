◆ inet_addr()

in_addr_t inet_addr ( const char *  cp)

将一个点分十进制的IP转换成一个长整数型数（u_long类型）

参数
[in]cp- 字符串，一个点分十进制的IP地址
返回
如果正确执行将返回一个无符号长整数型数。如果传入的字符串不是一个合法的IP地址，将返回INADDR_NONE

◆ inet_ntoa()

char* inet_ntoa ( struct in_addr  in)

将一个32位网络字节序的二进制IP地址转换成相应的点分十进制的IP地址

参数
[in]in- 一个表示Internet主机地址的结构
返回
若无错误发生，inet_ntoa()返回一个字符指针。否则的话，返回NULL
