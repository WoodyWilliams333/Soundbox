◆ file_type

uint8_t desfire_file::file_type

编码类型（这里为0x01）：
0x00 — 标准数据文件
0x01 — 备份数据文件
0x02 — 带备份的值文件
0x03 — 带备份的线性记录文件
0x04 — 带备份的循环记录文件
