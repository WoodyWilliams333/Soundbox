◆ font_get_word_info_user()

int font_get_word_info_user ( const font_t  pfont,
const char *  lpText,
int *  width,
int *  height 
)

获取字符(ASCII字符或汉字)在指定字库下的宽度和高度

参数
[in]pfont- 打开字库的操作句柄
[in]lpText- 传入的字符的起始地址，可以是单个ASCII字符或一个汉字
[out]width- 返回字符的宽度值
[out]height- 返回字符的高度值
返回
0 - 成功
<0 - 失败，返回相应的返回值

◆ font_open_user()

int font_open_user ( const char *  fontname,
const char *  charset,
int  style,
int  size,
font_t *  ppfont 
)

打开用户层的字库

参数
[in]fontname- 字库存放在文件系统中的绝对路径
[in]charset- 字库使用的字符集，见字符集的宏定义
[in]style- 字库的类型
[in]size- 字体的大小
[out]ppfont- font指针的指针，用于输出字库打开后的操作指针句柄
返回
0 - 成功
<0 - 失败，返回相应的返回值
