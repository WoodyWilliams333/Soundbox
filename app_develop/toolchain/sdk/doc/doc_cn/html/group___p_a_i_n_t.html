◆ paint_draw_text_rect_user()

int paint_draw_text_rect_user ( void *  fd,
int  nX,
int  nY,
void *  rect,
const char *  lpText,
Paint_t  paint 
)

在用户创建的画布上画字符串，该画字符串区域被限制在一个参数有rect指定矩形框内

参数
[in]fd- 打开字库取的的有效字库操作句柄
[in]nX- 起始位置的x坐标，可传负值
[in]nY- 起始位置的y坐标，可传负值
[in]rect- 矩形框参数 参考textRect
[in]lpText- 字符串内容
[in]paint- paint create user返回的paint
返回
int
  • =0 成功
  • <0 失败，返回相应的返回值
注解
  • 注意该接口是限制在一个矩形框内显示，不会对矩形框外的区域打点，矩形框坐标只能在屏幕坐标范围内，当坐标传为（0，x）， 参数nX为负数时，可以显示在最左边，y坐标类似。
