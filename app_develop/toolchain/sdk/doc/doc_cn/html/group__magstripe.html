详细描述

此模块用于描述 磁条卡模块 相关的接口

支持三个磁道的非阻塞式读取，读取出来的数据会在3S内清空以保证安全性

打开磁卡设备

参数
[in]filename- 磁卡解码设备名：“/dev/magcard0”,“/dev/magcard1”……
[in]oflags- 打开设备的模式，阻塞模式: O_RDONLY ,非阻塞模式:(O_RDONLY | O_NONBLOCK)
返回
打开的设备文件描述符
>= 0 - 成功,返回值用于操作磁卡解码设备的文件号
< 0 - 失败
