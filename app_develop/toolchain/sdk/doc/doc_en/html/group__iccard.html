◆ iccard_exchange_apdu()

int iccard_exchange_apdu ( int  fd,
uint32_t  sendlen,
const void *  senddata,
uint32_t *  recvlen,
void *  recvdata 
)

Send C-APDU to ICCard, and receive R-APDU from ICCard.

Parameters
[in]fd
  • device id, return from iccard_open_cardslot
[in]sendlen
  • length of c-apdu information
[in]senddata
  • c-apdu information
[out]recvlen
  • length of r-apdu information from iccard
[out]recvdata
  • r-apdu information from iccard
Returns

  • 0 success
  • -1 failed, please check "errno" for actual error code
  • ENXIO card not in slot
  • EINVAL invalid argument
  • ETIME timeout
  • ECOMM parity error or edc error
  • EPROTO invalid information from iccard
  • EACCES card need be reset first

int iccard_power_up ( int  fd,
int  vcc,
int  mode,
int *  len,
void *  atr 
)

reset cpucard and get the atr information

Parameters
[in]fd- device id, return from iccard_open_cardslot
[in]vcc- card operation vcc value:
  • CVCC_1_8_VOLT --— 1.8v voltage
  • CVCC_3_VOLT --— 3v voltage
  • CVCC_5_VOLT --— 5v voltage
[in]mode- card operation mode:
  • ISO_MODE --— iso7816 mode
  • EMV_MODE --— EMV specification mode
  • ISO_19200 --— higher rate iso7816 mode, 19200bps
  • ISO_38400 --— higher rate iso7816 mode, 38400bps
[out]len- length of atr information
[out]atr- atr information
Returns

0 -success
-1 -failed, please check "errno" for actual error code
ENXIO -card not in slot
EINVAL -invalid argument
ETIME -timeout
ECOMM -parity error or edc error
EPROTO -invalid information from iccard
