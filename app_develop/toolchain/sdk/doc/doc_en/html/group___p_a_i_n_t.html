const char *  lpText,
Paint_t  paint 
)

Draws a string on a user-created canvas. The string area is confined to a rectangle with rect specified as the parameter.

Parameters
[in]fd- Open a valid font operation handle from the font
[in]nX- The x coordinate of the starting position, which can pass negative values
[in]nY- The y coordinate of the starting position, which can pass negative values
[in]rect- Rectangle box parameter, referencetextRect
[in]lpText- String content
[in]paint- paint create user return paint value
Returns
int
  • =0 Success
  • <0 Fail
Note
  • Note that this interface is limited to a rectangular box display, does not dot the area outside the rectangular box, rectangular box coordinates can only be within the screen coordinates, when the coordinate is passed to (0, x), If the parameter nX is negative, it can be displayed on the far left, and the y coordinates are similar。
